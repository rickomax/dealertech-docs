# `vector.dot(a, b) -> number`

Returns the dot product of `a` and `b`.

Returns `1` if the vectors point the same way, `0` if perpendicular, `-1` if opposite (assuming both are normalized).

**Parameters**
- `a` (`vector`)
- `b` (`vector`)
