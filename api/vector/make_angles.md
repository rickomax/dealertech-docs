# `vector.make_angles(right, forward, up) -> vector`

Builds Euler angles (pitch / yaw / roll) from three basis vectors.

**Parameters**
- `right` (`vector`)
- `forward` (`vector`)
- `up` (`vector`)

**Returns**
- `vector` — The orientation as pitch / yaw / roll.
