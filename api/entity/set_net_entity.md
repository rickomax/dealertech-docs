# `entity:set_net_entity(key, value)`

Sets the entity referenced by a networked entity variable. Creates the variable if it does not already exist. Pass `nil` to clear the reference.

**Parameters**
- `key` (`string`)
- `value` (`entity|nil`)

**Server-only**
