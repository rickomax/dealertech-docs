# `vector4.create(x, y, z, w) -> vector4`

Creates a new 4D vector.

**Parameters**
- `x` (`number`) — X component (red channel).
- `y` (`number`) — Y component (green channel).
- `z` (`number`) — Z component (blue channel).
- `w` (`number`) — W component (alpha channel).

**Returns**
- `vector4`
