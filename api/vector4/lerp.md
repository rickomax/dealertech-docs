# `vector4.lerp(a, b, t) -> vector4`

Linearly interpolates between `a` and `b`.

**Parameters**
- `a` (`vector4`)
- `b` (`vector4`)
- `t` (`number`) — Interpolation factor, clamped to `[0, 1]`.
