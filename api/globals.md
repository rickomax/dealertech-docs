# `globals`

Global functions and variables registered in the Lua environment.

Functions are available immediately after the engine initializes. Runtime variables (`frame_time`, `time`, `me`) are updated each frame.

The class-table globals (`game`, `entity`, `vector`, etc.) are documented on their own pages under the API Reference.
