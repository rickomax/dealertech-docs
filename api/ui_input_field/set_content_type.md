# `ui_input_field:set_content_type(content_type)`

Sets the content validation mode for the input field.

**Parameters**
- `content_type` (`string`) — One of:
  - `"integer"`
  - `"decimal"`
  - `"password"`
  - `"email"`
  - any other value defaults to `"standard"`
