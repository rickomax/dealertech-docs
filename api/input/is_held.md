# `input.is_held(path) -> number`

Returns whether a raw control is currently being held down.

**Parameters**
- `path` (`string`) ‚Äî Raw control path (e.g. `<Keyboard>/space`).

**Returns**
- `number` ‚Äî `1` if held, `0` otherwise.
